**SmaC Tool**
================
Technological framework to facilitate the development of smart contracts.

**SmaC DSL**
_______________
Definition of a textual language (terminals, rules, patterns) for the construction of smart contracts using the Solidity language

Install SmaC from Update Site

If you have an Eclipse running :
Download the SmaC Update Site: 
Choose Help -> Install New Software... from the menu bar and click Add...
.
After a quick download and a restart of Eclipse
If you have an Eclipse running :

Choose Window -> Show View -> Other... from the menu bar
Select Xtext Tools -> Node Model Outline
The outline is added to your workbench
If you open an Xtext based file, the Node Model Outline displays the underlying Node Model
Integrating the Semantic Model Outline

Choose New -> Other -> SmaC Model from the menu bar
